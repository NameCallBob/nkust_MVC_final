111-1 MVC系統架構
=============

### 名稱：彬彬炸雞店

### 內容：  

線上訂購商品，且分為買賣家進行管理

  

目前以實踐的功能：

*   訂購功能
    
*   會員資料維護
    
*   商家資料維護
    
*   商品上下架
    
*   商品資料維護
    
*   訂單管理
    

  

使用前後端開發(前端JS+CSS+HTML。後端PHP+MYSQL)  
學習初階的MVC架構開發
